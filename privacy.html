Privacy Policy

Introduction

This privacy policy explains how the Tinder Auto Liker extension ("we", "our", or "the extension") handles your data.

Data Collection

The extension collects and stores the following information:

  • Total number of likes performed
  • Current session statistics (temporarily, in memory only)
Important: We do not collect, store, or transmit:
  • Personal information
  • Tinder profile data
  • Messages or matches
  • Browsing history
  • Any other personal data

Data Storage

All data is stored locally in your browser using Chrome's storage API. No data is sent to external servers.

Permissions

The extension requires the following permissions:

  • activeTab: To interact with the Tinder website
  • storage: To save your total likes count

Third-Party Services

The extension does not use any third-party services or analytics.

Data Deletion

You can clear all stored data by:

  • Uninstalling the extension
  • Clearing your browser data for the extension

Changes to Privacy Policy

We may update this privacy policy from time to time. Any changes will be reflected in the extension's updates.

Contact

If you have any questions about this privacy policy, please contact us through the Chrome Web Store.
